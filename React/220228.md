### 오늘 배운 것
- Recoil

### 새로 알게 된 것
- Recoil은 React를 위한 상태 관리 라이브러리이다.
- recoil 상태를 사용하는 컴포넌트는 부모 트리 어딘가에 나타내는 RecoilRoot가 필요하다.
- Atom은 상태의 일부를 나타내고, 어떤 변화가 있을 경우 구독하고 있는 모든 컴포넌트들이 재 렌더링 된다.
- recoil의 장점은 redux, mobx처럼 복잡한 상태구조를 가지고 있지 않고, 사용법 또한 간단하다.
